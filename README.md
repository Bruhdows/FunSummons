# FunSummons
Another Minecraft plugin made during Kasai's Testing Minecraft Plugins live stream.

My previous creation (during Kasai's stream): https://github.com/Bruhdows/GravityBlocks

And NO it is not a virus Kasai... haha. Although you kinda trusted me.

Created in about 2 hours.

## Controls
`Sneak+Swap Hands` - Opens accessories menu

## Commands
`/getitem <wand|accessory> <id>` - Get either wands or accessories

`/accessories` - Opens accessories menu
